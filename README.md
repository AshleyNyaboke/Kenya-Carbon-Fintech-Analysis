# Kenya FinTech & Climate NGO Data Analytics Pipeline

## Project Overview
This repository contains an end-to-end data analytics infrastructure tracking micro-payout efficiency, carbon accumulation rates (MTCO2e), and smallholder farmer retention metrics in Kenya. 

The pipeline simulates a green-finance framework where a climate NGO integrates with Safaricom's Daraja (M-Pesa) API to disburse mobile money incentives dynamically based on audited tree-survival metrics.

## Data Schema & Architecture
- **Fact Table:** `mpesa_carbon_payouts.csv` tracking localized data points across Kakamega, Nakuru, Nyeri, Meru, and Kilifi counties.
- **Key Dimensions:** Farmers, Regional Geographies, Calendar Timelines.
